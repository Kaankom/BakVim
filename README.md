# BakVim
The Chickentastic Neovim Config

```bash
git clone https://github.com/Kaankom/BakVim ~/.config/nvim
```
uses the following Plugins:
  - mason, mason-lspconfig, nvim-lspconfig, none-ls
  - treesitter
  - telescope
  - neo-tree
  - vim-tmux-navigator
  - lualine

Uses kanagawa as color schema
